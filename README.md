# 🎵 Harmonia Core API

API educacional de teoria musical estruturada por níveis de aprendizado.

---

## 📌 Sobre o Projeto

A **Harmonia Core API** é uma API REST desenvolvida em Python com Flask, criada com o objetivo de servir conteúdo estruturado de teoria musical dividido por níveis de aprendizado.

Este projeto faz parte do **Projeto Harmonia**, uma iniciativa voltada ao ensino musical organizado e progressivo.

Além do propósito educacional, este projeto também representa minha evolução prática no desenvolvimento backend, versionamento com Git e organização profissional de código.

---

## 🚀 Tecnologias Utilizadas

- Python
- Flask
- Git
- GitHub
- Ambiente Virtual (venv)

---

## 🧠 Objetivo da Versão 1.0

Nesta primeira versão, a API terá como foco:

- Estruturação de níveis de aprendizado
- Organização de módulos por nível
- Retorno de dados em formato JSON
- Estrutura preparada para futura integração com aplicações web e mobile

## 💻 Como Executar Localmente

1. Clone o repositório  
2. Crie e ative um ambiente virtual  
3. Instale as dependências:

```
pip install -r requirements.txt
```

4. Execute:

```
python app.py
```

A aplicação rodará em:

```
http://127.0.0.1:5000
```
